NOvid-20

An amazing PWA which is a solution to foretell the likelihood of being in vicinity of COVID-19 and has the following features:
  1. Prediction model*
    2. This would be used for providing the likelihood of the disease based on various factors.

  2. Interactive Chatbot
    3. This is an interactive bot which will resolve all your queries related to the disease and provide implementable solutions to them.

  3. Some useful insights
    4. In addition to the above, we would also be providing useful insights from our end, which would prove to be beneficial.

  4. The current situation's overview
    5. We would also be giving the latest updates and overview of the on-going situation to help you keep track of it.

Model Structure
This deployed model has four major parts :
  1. model.py - This contains code for our Machine Learning model to predict Covid19 on training data in 'Cleaned-data.csv' file.
  2. app.py - This contains Flask APIs that receives user input through GUI and computes the predicted value based on our model and returns it.
  3. templates - This folder contains the HTML template to allow user to enter their details and symptoms and displays the predicted output.
